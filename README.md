# «Ұлағат» оқырман клубы

**Кітап. Ой. Рух. Қауымдастық.**
Оқу мәдениетін қалыптастыратын және кітапсүйер қауымды біріктіретін заманауи орта.

---

## 📖 Жоба туралы

**«Ұлағат»** веб-қосымшасы — оқырман клубының негізгі сандық платформасы. Бұл сайт арқылы клубқа жаңа мүшелер қабылданады, оқылатын кітаптардың тізімі жарияланады, онлайн пікірталастар жүргізіледі және клубтың іс-шаралары мен жаңалықтары бөлісіледі.

### 🌟 Негізгі мүмкіндіктер мен бөлімдер:

- **Кітаптар мен пікірталас (`/books`):** Әр оқылған кітапқа арналған жеке виртуалды талқылау беттері.
- **«Ұлағат үні» ансамблі (`/ensemble`):** Клуб ішіндегі өнерлі жастар хақында ақпарат және олардың шығармашылығы.
- **Іс-шаралар мен кездесулер:** Алдағы жиындар туралы хабарландырулар.
- **Оқырмандарды қабылдау (`/apply`):** Жаңа мүшелердің веб-сайт арқылы өтінім қалдыру мүмкіндігі (Firebase арқылы жұмыс істейді).
- **Галерея мен Блог:** Клуб өмірінен естеліктер мен мақалалар жинағы.

## 🛠 Технологиялық стек (Tech Stack)

Жоба заманауи Frontend технологияларын пайдалана отырып жасалған:

- **Фреймворк:** [React 19](https://react.dev/) & [Vite](https://vitejs.dev/)
- **Бағыттау (Routing):** React Router v7
- **Стилизация:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Анимациялар:** [Framer Motion](https://www.framer.com/motion/)
- **Дерекқор мен Бэкенд:** [Firebase](https://firebase.google.com/) (Firestore)
- **Иконкалар:** [Lucide React](https://lucide.dev/)
- **Қосымша құралдар:** TypeScript, Express (локальды тестілеу үшін)

## 💻 Бағдарламаны іске қосу (Run Locally)

Жобаны өз компьютеріңізде көшіріп алып, іске қосу үшін келесі нұсқауларды орындаңыз:

### Талаптар (Prerequisites):

- [Node.js](https://nodejs.org/) (v18 немесе одан жоғары нұсқа) орнатылуы қажет.

### 1-қадам. Жобаны орнату

Терминал арқылы жобаның папкасына кіріп, барлық бумаларды жүктеп алыңыз:

```bash
npm install
```

### 2-қадам. Қоршаған ортаны реттеу (Environment Setup)

Егер сізде `.env` файлы болмаса, `.env.example` файлының көшірмесін жасап, атын `.env` (немесе `.env.local`) деп өзгертіңіз:

```bash
cp .env.example .env
```

### 3-қадам. Локальды серверді іске қосу

Жобаны әзірлеу (development) режимінде қосу үшін:

```bash
npm run dev
```

👉 [http://localhost:3000](http://localhost:3000) сілтемесін браузерден ашып, сайтты көре аласыз.

---

## 📦 Толық нұсқаны құрастыру (Build for Production)

Жобаны серверге (Hosting) жүктеуге дайындау үшін:

```bash
npm run build
```

Құрастырылған нұсқа `dist` папкасында пайда болады. Оны тексеріп көру үшін мына команданы қолданыңыз:

```bash
npm run preview
```
